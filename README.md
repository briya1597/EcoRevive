# Eco-Revive: Reuse, Recycle, Re-Think

Eco-Revive is a web-based educational platform dedicated to promoting sustainable waste management through the "6Rs" principles. The project aims to raise environmental consciousness by providing practical insights into reusing, recycling, and rethinking our consumption habits.

## 🌟 Features

- **Educational Content**: Detailed guides on how to unleash the power of second chances (Reuse), close the loop (Recycle), and innovate for tomorrow (Re-Think).
- **Interactive UI**: Engaging user interface with original background animations and a clean, responsive layout.
- **Global Navigation**: Seamless movement between pages with a persistent navigation bar.
- **Contact Us**: Built-in contact form for community engagement and team outreach.

## 📂 Project Structure

- `index.html`: The main landing page and entry point for the website.
- `Reuse.html`, `Recycle.html`, `ReThink.html`: Specialized content pages for each principle.
- `ContactUs.html`: Contact form and team member information.
- `assets/images/`: Centralized directory for all visual assets and media.

## 🚀 Deployment

This project is optimized for **GitHub Pages**.

### How to Deploy:
1. Push this repository to your GitHub account.
2. Navigate to the **Settings** tab.
3. Select **Pages** from the sidebar.
4. Under **Build and deployment**, set the source to `Deploy from a branch`.
5. Select the `main` branch and the `/(root)` folder.
6. Click **Save** and wait for your site to go live!
